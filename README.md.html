Entertainment Center

Entertainment Center is a a movie website based on some of my favorite movies. The website provide information such as:

  • Movie title
  • Movie Storyline
  • Movie poster
  • Movie embedded trailer URL

Modules

  • entertainment_center.py
    • instance of media.py
  • media.py
    • class definition for entertainment_center.py
  • fresh_tomatoes.py
    • generates a website that displays movies

Version

0.1

Installation

fresh_tomatoes.py needs to be in the same directory as media.py and entertainment_center.py.
